
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Implement Core Game Logic

#### Description
Create a Hangman game that manages word selection, letter guessing, and game state tracking.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses and show current progress in underscore format (`_ _ _`)
- Track incorrect guesses and remaining attempts
- Handle repeated guesses without subtracting additional attempts
- Display a win message when the word is guessed and a lose message when attempts run out

### 🛠️ User Interaction and Feedback

#### Description
Add clear prompts and game feedback to help the player understand the game state.

#### Requirements
Completed program should:

- Prompt the player for a letter guess each turn
- Show the current word state after each guess
- Display letters already guessed
- Inform the player when a guess is correct or incorrect
- End with a final message indicating whether the player won or lost
